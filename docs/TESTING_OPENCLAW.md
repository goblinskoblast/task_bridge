# Тестирование OpenClaw интеграции

Полный набор тестов для проверки работы OpenClaw как основного AI агента в TaskBridge.

---

## 🎯 Что тестируем

1. **Извлечение задач из Telegram сообщений**
2. **Извлечение задач из Email**
3. **Support chat (AI консультант)**
4. **Vision API (анализ скриншотов)**
5. **Производительность и качество**

---

## Тест 1: Извлечение задач из Telegram

### 1.1 Простая задача

**Отправьте в группу:**
```
@username сделай отчет до завтра
```

**Ожидаемый результат:**
- ✅ Бот распознал задачу
- ✅ Извлек assignee: @username
- ✅ Извлек due_date: завтра
- ✅ Создал задачу или pending task
- ✅ Отправил уведомление

**Проверка качества:**
- Правильно ли определен дедлайн?
- Корректное ли описание задачи?

---

### 1.2 Сложная задача с множественными параметрами

**Отправьте:**
```
@user1 @user2 срочно подготовьте презентацию для клиента до пятницы 15:00, очень важно
```

**Ожидаемый результат:**
- ✅ Несколько assignees (user1, user2)
- ✅ Priority: urgent или high
- ✅ Due date: пятница 15:00
- ✅ Правильное описание

---

### 1.3 Неявная задача (тест на умность AI)

**Отправьте:**
```
@username не мог бы ты посмотреть на баг в production к концу недели?
```

**Ожидаемый результат:**
- ✅ Распознал как задачу (вежливая просьба)
- ✅ Assignee: @username
- ✅ Due date: конец недели
- ✅ Priority: возможно high (production bug)

**Сравнение с OpenAI:**
- OpenClaw должен лучше понимать контекст
- Более точное определение приоритета

---

### 1.4 НЕ задача (негативный тест)

**Отправьте:**
```
Кто-нибудь знает где кофе?
```

**Ожидаемый результат:**
- ✅ Бот НЕ создал задачу
- ✅ Проигнорировал сообщение

## Что делать если тесты провалились

### Проблема: Медленные ответы (>10 сек)

**Решение:**
1. Увеличьте timeout: `OPENCLAW_TIMEOUT=120`
2. Проверьте интернет соединение
3. Рассмотрите платный ngrok (меньше latency)

---

### Проблема: Качество хуже чем OpenAI

**Решение:**
1. Попробуйте другую модель: `OPENCLAW_MODEL=anthropic/claude-sonnet-3.5`
2. Настройте температуру в prompts
3. Улучшите system prompts

---

### Проблема: Частые timeout/ошибки

**Решение:**
1. Проверьте OpenClaw логи на ошибки
2. Перезапустите OpenClaw: `docker-compose restart`
3. Проверьте OpenAI API key валиден
4. Добавьте fallback на OpenAI

## Автоматизированное тестирование (опционально)

Можно создать тестовый скрипт:

```python
# test_openclaw.py
import asyncio
from bot.ai_extractor import analyze_message_with_ai

async def test_all():
    # Тест 1: Простая задача
    result = await analyze_message_with_ai("@user сделай отчет до завтра")
    assert result['has_task'] == True
    print("✅ Тест 1 пройден")

    # Тест 2: Не задача
    result = await analyze_message_with_ai("Где кофе?")
    assert result['has_task'] == False
    print("✅ Тест 2 пройден")

    # ... добавьте больше тестов

if __name__ == "__main__":
    asyncio.run(test_all())
```

Запуск:
```bash
python test_openclaw.py
```
